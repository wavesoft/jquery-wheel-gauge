jquery-wheel-gauge

In this page we are demonstrating the use of jquery-wheel-gauge plug-in that displays values in a bit more retro spinning-wheel way.

Simple drop-in replacement

The wheel gauge plug-in can be used as drop-in replacement to your input element.

Sample Javascript 

$("input.dial.simple").rundial();

// Change values like this:
$("input.dial.simple")
	.val(50).trigger('change');

With styling information

You can optionally specify customization parameters.

Sample Javascript 

$("input.dial").rundial({
    width: 80,
    height: 80
});

Range and steps

You can define what's the value range and subdivision steps.

Sample Javascript 

$("input.dial.subdiv").rundial({
    width: 80,
    height: 80,
    min: -500,
    max: 500,
    step: 20
});

You can also reverse the order

You can define what's the value range and subdivision steps.

Sample Javascript 

$("input.dial.reverse").rundial({
    width: 80,
    height: 80,
    ascending: true
});

Custom format

If you want to display different values, you can use a formatting function.

Sample Javascript 

$("input.dial.custom").rundial({
    width: 160,
    height: 80,
    format: function(v) { return ""+v+"%"; }
});

Out of range customization

The wheel widget will automatically clip the out-of-range values. The way this happens is fully customizable.

Sample Javascript 

$("input.dial.oor").rundial({
    width: 80,
    height: 80,
    min: 10,
    max: 20,
    overflowText: "-- top --",
    overflowElmClass: "rundial-warn",
    overflowClass: "rundial-overrun",
    underflowText: "-- bot --",
    underflowElmClass: "rundial-warn",
    underflowClass: "rundial-underrun"
});

Setting values

The wheel widget will use the iniital value of the input field. Any further change should be triggered using the .rundial() interface.

Sample Javascript 

$("input.api-dial").rundial({
    width: 80,
    height: 80,
});

$("#btn-shuffle").click(function() {
    $("input.api-dial").rundial('value', Math.random()*100);
});
